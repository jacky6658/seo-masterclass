# SEO Masterclass — 艾澤拉斯搜尋之戰

互動式 SEO / AEO / GEO 學習教材，用魔獸世界觀講故事。

**直接用瀏覽器開啟 `seo-masterclass.html` 即可閱讀，支援手機 RWD。**

---

## 4 天學習路徑

| Day | 主題 | 內容 |
|-----|------|------|
| **Day 1** | Algorithm | Google 演算法底層邏輯、五大核心系統、2025-2026 更新時間線 |
| **Day 2** | Factions | 白帽 / 灰帽 / 黑帽技術全解析、Decision Matrix |
| **Day 3** | AI Era | SEO + AEO + GEO 三層策略、RAG Pipeline、6 Phase 副本攻略 |
| **Day 4** | Advanced | 灰色產業 SEO、季節性攻勢（世界盃）、UI/UX 與排名、AI 輔助工作流、進階技術 |

## 互動功能

- Tab 切換 4 天內容
- Accordion 展開 / 收合章節
- 每天結束有 Quiz 測驗（即時回饋）
- Achievement 成就彈窗
- Action Checklist（進度自動存檔至 localStorage）
- 閱讀進度條
